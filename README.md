# AzuriomInstaller

[![Chat](https://img.shields.io/discord/625774284823986183?color=7289da&label=Discord&logo=discord&logoColor=fff&style=flat-square)](https://azuriom.com/discord)

An automatic installer for [Azuriom](https://azuriom.com/) built with [Vue.js](https://vuejs.org/).

## Project setup for development
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
